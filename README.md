
## Express React.JS Isomorphic Todolist Test Project

#### Run on Cloud9  https://c9.io

#### Use

1) Express 4.13.1

2) Mongodb

3) React 0.13.3

4) Webpack


#### Running the server

1) Installation

    $ npm install

1) Build ReactJS

    $ npm install -g webpack
    $ webpack --watch

2) Alternatively you can launch the app from the Terminal:
    
    $ npm run dev
